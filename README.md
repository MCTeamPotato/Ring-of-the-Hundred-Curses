# 玩游戏时在做什么？有没有空？可以加上百咒吗？

![ring.png](img%2Fring.png)

---

## 起因
你游玩家特有的抖M属性喜欢玩困难生存，七咒之戒便是最常见的“嘉宾”之一。  
于是为了让玩家的游戏体验更加的快乐我们便想出来了《百咒之戒》这个东西。（😋）
